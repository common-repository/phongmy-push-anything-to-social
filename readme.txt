#This's plugins help Owner push order from Woocommerce to Facebook messenger quickly base On CallmeBot API

== Description ==
#This's plugins help Owner push order from Woocommerce to Facebook messenger quickly base On CallmeBot API
# Support send to Facebook messager for both admin and Technical or customer.

# Notify send to facebook when:
- Have new order.
- Order completed.
- Order cancelled.
- Order refund.

# Variable available in Plugins
- Your order ID: #%%order_id%%
- Products name: %%product_name%%
- First name: %%first_name%%
- Last name: %%last_name%%
- Customer email: %%billing_email%%
- Phone number: %%billing_phone%%
- Address: %%billing_address%%
- Payment methods: %%payment_method%%
- Delivery method: %%shipping_method%%
- New Order date: %%created_date%%
- Completed Order date: %%completed_date%%
- Customer note: %%customer_note%%
- Total money: %%total%%


# Create Callmebot API
- Login facebook.com and chat with BOT - https://m.me/api.callmebot
- type: "create apikey" to get API KEY for free


== Installation ==
1. Find plugin with name "Push Anything To Social".
2. Active plugin after install.
3. Go to Settings -> Anything to social and config API.
